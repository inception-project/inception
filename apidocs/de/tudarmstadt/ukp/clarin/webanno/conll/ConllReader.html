public class ConllReader
extends de.tudarmstadt.ukp.dkpro.core.api.io.JCasResourceCollectionReader_ImplBase

Reads a specific Conll File (9 TAB separated) annotation and change it to CAS object. Example of Input Files: 1 Heutzutage heutzutage ADV _ _ 2 ADV _ _ First column: token Number, in a sentence second Column: the token third column: the lemma forth column: the POS fifth/sixth xolumn: Not Yet known seventh column: the target token for a dependency parsing eighth column: the function of the dependency parsing ninth and tenth column: Not Yet Known Sentences are separated by a blank new line
