This telemetry collector helps the developer team of discover the number of installations and the size of the user base. These are important indicators for the success of the product and are e.g. included in reports to funders. Please support the future development of by submitting these anonymous usage statistics to the developers at UKP Lab, Technische Universität Darmstadt, Germany.

The anonymous usage statistics are sent to our server every time the application is started.

Use the Toogle details... button above to discover which data we collect and why.
