[IMPORTANT]
====
Keyboard shortcuts should always include modifier keys (Ctrl, Shift, Alt) to:

* Work properly even when focus is in input fields (modifier keys prevent conflicts with normal typing)
* Avoid conflicts with standard browser shortcuts (e.g., use Ctrl+y instead of just y)
====

NOTE: Keyboard shortcuts are specified as comma-separated key names (e.g., `Ctrl,z` for Ctrl+Z). Available key names include: `Ctrl`, `Shift`, `Alt`, `Meta`, `Enter`, `Escape`, `Delete`, `Backspace`, `Tab`, `Space`, `Page_up`, `Page_down`, `Home`, `End`, `Insert`, arrow keys (`Up`, `Down`, `Left`, `Right`), function keys (`F1` through `F12`), and regular character keys (`a` through `z`, `0` through `9`).
