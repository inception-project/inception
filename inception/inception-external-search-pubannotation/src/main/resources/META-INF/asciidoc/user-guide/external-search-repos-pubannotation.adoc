= PubAnnotation

link:http://www.pubannotation.org[PubAnnotation] is a repository through which anyone can share
their texts and annotations with others. It can be added as an external document repository by
selecting the **PubAnnotation** repository type.

== Configuration

The repository has two configuration fields:

* **URL**: the base URL of the PubAnnotation server (default: `https://pubannotation.org`).
* **Project**: optional. When set, search results and document fetches are scoped to a single
  PubAnnotation project. Annotations imported from a scoped repository follow the project's
  schema (e.g. `PubmedHPO` annotates HPO ontology IDs, `GO-BP` annotates Gene Ontology IRIs).
  Use the **Browse...** button next to the field to pick a project from a searchable list of
  all known PubAnnotation projects, complete with maintainer, author, license, and last-updated
  metadata.

NOTE: PubAnnotation does not formally describe a project's annotation scheme via the API. A
      project's free-text description on its public page often hints at the vocabulary used,
      and you can preview an annotated document at
      `https://pubannotation.org/projects/<project>/docs/sourcedb/<db>/sourceid/<id>` to see
      the entity types in use.

== Searching

In the external search panel:

* Enter free-text keywords to search by content.
* Or use the syntax `id:<sourcedb>/<sourceid>` (e.g. `id:PubMed/25314077`) to fetch a single
  specific document directly without keyword search.

When a **Project** is configured on the repository, only documents annotated by that project
are returned.

== Importing

Imported documents arrive as `pubannotation-annotations` documents — text plus all of the
project's denotations, relations, and attributes. The exact mapping from PubAnnotation
annotations to {product-name} layers and features is described in
<<sect_formats_pubannotation>>.
