[cols="2,1,1,3"]
|====
| Format | Import | Export | Supported layers

| Perseus Ancient Greek and Latin Dependency Treebank 2.1 XML (`perseus_2.1`)
| yes
| no
| Part-of-speech tagging (built-in), +
  Lemmatization (built-ion), +
  Dependency parsing (built-in)
|====
