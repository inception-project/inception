Units
  •   Agreement
    There is no disagreement between annotators. There is nothing to curate.
  •   Curated
    For every position any annotator has annotated, the is a corresponding annotation at the same position in the curator's document.
  •   Stacked annotations
    A single annotator has made multiple annotations of the same layer at a given position.
  •   Incomplete annotation
    A position has been annoated by some annotators but not by all.
  •   Disagreement
    Multiple annotators have annotated a position, but different labels were assigned.
